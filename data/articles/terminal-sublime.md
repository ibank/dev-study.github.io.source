1. 어제 이강훈님께서 보여주신 터미널에서 Sublime Text 2 켜는
   명령어입니다.

```sh
subl .
```

2. bash 에서 subl 커멘드를 못찾았다고 나오시는 분은

```sh
sudo ln -s /Applications/Sublime\ Text\ 2.app/Contents/SharedSupport/bin/subl /usr/bin/subl
```

3. 입력 후 실행
